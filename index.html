Sincronización Cloud Firebase

Para sincronizar tus datos en la nube (ideal para GitHub Pages), necesitas crear un proyecto gratuito en Firebase e ingresar tu configuración aquí:

¿Cómo obtenerlo?
1. Entra a Firebase Console y crea un proyecto.
2. Agrega una "Web App" al proyecto.
3. Habilita Cloud Firestore y Anonymous Authentication (o Email/Password).
4. Copia el objeto `firebaseConfig` que te darán y pégalo arriba.
